# Custom Contact Form Plugin

A lightweight WordPress contact form plugin built from scratch.

## Features

* Contact Form Shortcode
* Admin Inbox
* Read / Unread Messages
* Search Messages
* Pagination
* Bulk Delete
* Email Notifications
* Export Messages to CSV
* Settings Page
* Spam Protection (Honeypot)

## Shortcode

```text
[custom_contact_form]
```

## Technologies Used

* PHP
* WordPress Plugin API
* MySQL
* HTML
* CSS
* JavaScript
