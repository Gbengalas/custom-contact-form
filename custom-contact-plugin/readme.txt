Custom Contact Form

Version: 1.0.0

Description
A lightweight WordPress contact form plugin with inbox management, email notifications, CSV export, spam protection, and settings.

Features
- Contact Form Shortcode
- Admin Inbox
- Read / Unread Messages
- Search Messages
- Pagination
- Bulk Delete
- Email Notifications
- Export Messages (CSV)
- Settings Page
- Spam Protection (Honeypot)
- Contact Form Guide

Shortcode

[custom_contact_form]

Installation

1. Upload the plugin to wp-content/plugins/
2. Activate the plugin
3. Create a page
4. Add the shortcode:
   [custom_contact_form]
5. Publish the page
